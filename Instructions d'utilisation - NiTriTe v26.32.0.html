Prérequis & Compatibilité

  • Système d'exploitation : Windows 10 (build 1903+) ou Windows 11 (recommandé)
  • Architecture : 64 bits uniquement (x86-64)
  • Droits requis : Administrateur — UAC obligatoire au lancement
  • WebView2 Runtime : Inclus nativement dans Windows 11 · Windows 10 : installé automatiquement si absent
  • RAM recommandée : 4 GB minimum, 8 GB recommandé
  • Espace disque : ~50 MB (Version 1) · ~3 GB (Versions 2 et 3)
⚠
NiTriTe requiert les droits Administrateur pour accéder aux informations système bas niveau, modifier les services, le registre et les pilotes. Le bouclier UAC s'affiche au lancement. Sans droits admin, de nombreuses fonctions seront limitées ou inaccessibles.

Les 3 versions portables — Comment lancer NiTriTe

Version 1 — EXE Portable seul
Nitrite_v26.32.0_portable.exe
~15 MB

L'intégralité de NiTriTe est compilée dans un seul fichier exécutable. Le frontend Vue 3, le moteur Rust et les manifests UAC sont embarqués. Les dossiers logiciel\, Drivers\ et Script Windows\ ne sont pas inclus.

Lancement — Étapes
  1. Double-cliquez sur Nitrite_v26.32.0_portable.exe
  2. La fenêtre UAC s'affiche → cliquez "Oui" pour accorder les droits admin
  3. NiTriTe s'ouvre instantanément — aucune installation, aucun fichier créé
Fonctions disponibles
✓ Tableau de bord complet
✓ Diagnostic système (47 onglets)
✓ Optimisations (debloat, réseau)
✓ Benchmark CPU/RAM/Disque
✓ Nettoyeur, BSOD, Boot Manager…
✓ Réseau, Terminal, Agent IA
✗ Apps Portables (logiciel\ absent)
✗ Drivers (Drivers\ absent)
✗ Scripts Windows (absent)
💡
Idéal pour : diagnostic rapide, transport sur clé USB légère. Si vous avez besoin des portables/drivers/scripts, déplacez Nitrite.exe dans un dossier contenant logiciel\, Drivers\ et Script Windows\ — ils seront automatiquement détectés.
Version 2 — Dossier portable complet
Nitrite_v26.32.0\
~2.4 GB
Structure du dossier
Nitrite_v26.32.0\ ├── Nitrite.exe ← Application principale (15 MB) ├── logiciel\ ← 18 outils portables (Autoruns, CPU-Z…) ├── Drivers\ ← Pilotes (Realtek, runtimes Windows…) └── Script Windows\ ← Scripts tweaks et optimisation Windows
Lancement — Étapes
  1. Copiez le dossier Nitrite_v26.32.0\ entier sur votre support (clé USB, disque externe, Bureau…)
  2. Ouvrez le dossier
  3. Double-cliquez sur Nitrite.exe
  4. Acceptez l'UAC (droits admin)
  5. NiTriTe s'ouvre avec accès à la totalité des fonctions
⚠
Ne déplacez pas Nitrite.exe seul hors du dossier. Les chemins vers logiciel\, Drivers\ et Script Windows\ sont relatifs à l'emplacement de Nitrite.exe.
Version 3 — SFX tout-en-un (auto-extractible)
Nitrite_v26.32.0_full.exe
~1.2–1.5 GB compressé

Tout le contenu de la Version 2 compressé dans un unique exécutable. Nitrite.exe + logiciel\ + Drivers\ + Script Windows\ en un seul fichier.

Première utilisation (extraction)
  1. Double-cliquez sur Nitrite_v26.32.0_full.exe
  2. Acceptez l'UAC
  3. Cliquez "Oui" pour confirmer l'extraction
  4. Choisissez le dossier de destination
  5. L'extraction commence — durée estimée : 3–15 min selon la vitesse du disque
  6. Une fois terminé, Nitrite.exe se lance automatiquement
Lancement suivant (après extraction)
  • Naviguez jusqu'au dossier d'extraction → double-cliquez sur Nitrite.exe
  • Le fichier _full.exe n'est plus nécessaire — vous pouvez le supprimer
⚠
Espace requis : Prévoyez ~3 GB d'espace libre sur le disque cible. Clé USB 2.0 déconseillée pour l'extraction (30–60 min).

Résolution de problèmes

⚠ "localhost a refusé la connexion" / page blanche ERR_CONNECTION_REFUSED ▾
Cause : Vous utilisez un exe de développement sans frontend embarqué.
  • Utilisez UNIQUEMENT les exes dans release\ : Nitrite_v26.32.0_portable.exe, la Version 2, ou la Version 3 après extraction
  • Les fichiers dans src-tauri\target\release\ sont réservés au développement
⚠ L'application ne démarre pas (rien ne se passe au double-clic) ▾
Cause : UAC refusé, ou WebView2 Runtime absent (rare sur Windows 11).
  • Clic droit sur Nitrite.exe → "Exécuter en tant qu'administrateur"
  • Si le problème persiste : téléchargez WebView2 Runtime depuis microsoft.com
⚠ Les Apps Portables / Drivers / Scripts sont absents ▾
Cause : Vous utilisez la Version 1 sans les dossiers logiciel\, Drivers\, Script Windows\.
  • Utilisez la Version 2 ou 3 qui inclut tous ces dossiers
  • Ou placez manuellement ces dossiers dans le même répertoire que Nitrite.exe
⚠ L'Agent IA ne répond pas / "Impossible de contacter Ollama" ▾
Cause : Ollama n'est pas installé, démarré, ou le modèle est absent.
  1. Installez Ollama : https://ollama.ai
  2. Terminal : ollama pull llama3:8b
  3. Démarrez : ollama serve (vérifiez sur http://localhost:11434)
  4. NiTriTe → Paramètres → IA : vérifiez URL et nom du modèle
⚠ "Accès refusé" sur certaines fonctions ▾
Cause : NiTriTe n'a pas été lancé avec les droits Administrateur.
  • Clic droit → "Exécuter en tant qu'administrateur"
  • Vérifiez que votre compte Windows est bien Administrateur
⚠ L'interface est blanche ou ne charge pas ▾
