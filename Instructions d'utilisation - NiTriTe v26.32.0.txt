1. PRÉREQUIS & COMPATIBILITÉ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Système d'exploitation : Windows 10 (build 1903+) ou Windows 11 (recommandé)
  Architecture           : 64 bits uniquement (x86-64)
  Droits requis          : Administrateur — UAC obligatoire au lancement
  WebView2 Runtime       : Inclus nativement dans Windows 11
                           Windows 10 : installé automatiquement si absent
  RAM recommandée        : 4 GB minimum, 8 GB recommandé
  Espace disque          : ~50 MB pour l'application seule (Version 1)
                           ~3 GB pour la version complète (Versions 2 et 3)

  ⚠  NiTriTe requiert les droits Administrateur pour accéder aux informations
     système bas niveau, modifier les services, le registre, les pilotes et
     exécuter les outils de maintenance. Le bouclier UAC s'affiche au lancement.
     Sans droits admin, de nombreuses fonctions seront limitées ou inaccessibles.

║  VERSION 1 — EXE Portable seul                                             ║
║  Fichier : Nitrite_v26.32.0_portable.exe   (~15 MB)                        ║
╚═════════════════════════════════════════════════════════════════════════════╝

  CONTENU :
    L'intégralité de NiTriTe est compilée dans un seul fichier exécutable.
    Le frontend Vue 3, le moteur Rust et les manifests UAC sont embarqués.
    Les dossiers logiciel\, Drivers\ et Script Windows\ ne sont PAS inclus.

  LANCEMENT — ÉTAPES :
    1. Double-cliquez sur  Nitrite_v26.32.0_portable.exe
    2. La fenêtre UAC s'affiche → cliquez "Oui" pour accorder les droits admin
    3. NiTriTe s'ouvre instantanément — aucune installation, aucun fichier créé

  FONCTIONS DISPONIBLES AVEC CETTE VERSION :
    ✓ Tableau de bord complet (monitoring temps réel)
    ✓ Diagnostic système (47 onglets)
    ✓ Optimisations (debloat, réseau, performances)
    ✓ Benchmark (CPU, RAM, Disque)
    ✓ Nettoyeur avancé, Récupération données, BSOD, Boot Manager…
    ✓ Réseau, Terminal, Agent IA, Paramètres, Profils
    ✗ Apps Portables (logiciel\ absent)
    ✗ Drivers (Drivers\ absent)
    ✗ Scripts Windows (Script Windows\ absent)

  IDÉAL POUR :
    → Diagnostic rapide d'un PC sans outils externes
    → Transport sur clé USB très légère (15 MB seulement)
    → Environnements où l'espace disque est très limité

  ⚠  Si vous avez besoin des portables/drivers/scripts, déplacez Nitrite.exe
     dans un dossier qui contient les sous-dossiers logiciel\, Drivers\ et
     Script Windows\ — ils seront automatiquement détectés.

    ├── Drivers\                 ← Pilotes (Realtek, runtimes Windows…)

  LANCEMENT — ÉTAPES :
    1. Copiez le dossier  Nitrite_v26.32.0\  entier sur votre support

    2. Ouvrez le dossier
    3. Double-cliquez sur  Nitrite.exe
    4. Acceptez l'UAC (droits admin)
    5. NiTriTe s'ouvre avec accès à la totalité des fonctions

  TOUTES LES FONCTIONS SONT DISPONIBLES :
    ✓ Tout ce que la Version 1 propose
    ✓ Apps Portables (18 outils lancables depuis NiTriTe)
    ✓ Installation de Drivers depuis le dossier Drivers\
    ✓ Bibliothèque de Scripts Windows

  ⚠  IMPORTANT — Ne déplacez pas Nitrite.exe seul hors du dossier.
     Les chemins vers logiciel\, Drivers\ et Script Windows\ sont relatifs
     à l'emplacement de Nitrite.exe. Si vous déplacez uniquement l'exe,
     ces sections ne fonctionneront plus.

  IDÉAL POUR :
    → Techniciens informatiques en déplacement (clé USB de dépannage complète)
    → Utilisation régulière sur plusieurs PC clients
    → Accès à tous les logiciels portables intégrés sans installation

       ┌─────────────────────────────────────────────┐
       │  Bienvenue dans Nitrite v26.32.0             │
       │  Portable Setup                              │
       │                                              │
       │  [Suivant >]                                 │

    5. Choisissez le dossier de destination :
       • Par défaut :  C:\Users\[VotreNom]\AppData\Local\Nitrite_v26.32.0\

    6. Cliquez [Installer] — l'extraction commence
       ⏱ Durée estimée : 3 à 15 minutes selon la vitesse du disque cible
    7. Une fois terminé :
       • Cochez "Lancer Nitrite maintenant"
       • Cliquez [Terminer]
       → NiTriTe se lance automatiquement

  LANCEMENT SUIVANT (après extraction) :
    → Naviguez jusqu'au dossier d'extraction choisi
    → Double-cliquez sur  Nitrite.exe
    Le fichier _full.exe n'est PLUS nécessaire une fois extrait.
    Vous pouvez le supprimer pour libérer de l'espace.

  IDÉAL POUR :
    → Partager NiTriTe complet en un seul fichier (clé USB, serveur, email)
    → Déploiement rapide sur un nouveau PC
    → Archivage d'une version complète horodatée de la suite

  ⚠  ESPACE REQUIS : Prévoyez ~3 GB d'espace libre sur le disque cible.
     L'extraction est définitive — le dossier extrait est la Version 2.

┌──────────────────────────────────────────────────────────────────────────────
│  TABLEAU DE BORD   /
└──────────────────────────────────────────────────────────────────────────────

  Page d'accueil. Vue d'ensemble de la santé du système en temps réel.

  INFORMATIONS AFFICHÉES :
    • Score de santé global  (0-100) — évaluation automatique de l'état du PC
    • Sparklines 20 points   — graphiques miniatures CPU / RAM / Disque / Réseau
    • Top 8 processus        — les plus gourmands en CPU et RAM
    • CPU                    — usage %, fréquence, température (si capteur dispo)
    • RAM                    — total / utilisé / disponible en GB
    • GPU                    — modèle, usage %, VRAM, température
    • Disque                 — espace utilisé par volume, activité I/O
    • Réseau                 — débits montant/descendant en temps réel (KB/s ou MB/s)
    • Uptime                 — durée depuis le dernier démarrage
    • OS                     — version Windows + build number + architecture

  ACTIONS RAPIDES (boutons sur le dashboard) :
    [Diagnostic]   → va directement au Diagnostic complet
    [Nettoyage]    → va au Nettoyeur Avancé
    [Mises à jour] → va aux Mises à jour
    [Sauvegarde]   → va à la Sauvegarde
    [Antivirus]    → va au Scan Antivirus

  Toutes les données se rafraîchissent automatiquement toutes les 1-5 secondes
  selon l'intervalle configuré dans Paramètres.


┌──────────────────────────────────────────────────────────────────────────────
│  DIAGNOSTIC   /diagnostic
└──────────────────────────────────────────────────────────────────────────────

  Module d'analyse système complet en 47 onglets, regroupés en 6 catégories
  filtrables via les "pills" en haut de page.

  ── [MATÉRIEL] ─────────────────────────────────────────────────────────────

  CPU
    • Modèle complet, fabricant, architecture
    • Fréquence de base et boost, nombre de cœurs physiques / logiques
    • Cache L1/L2/L3, TDP
    • Usage % en temps réel par cœur individuel
    • Température (si capteur disponible)

  GPU
    • Modèle et fabricant, VRAM totale / utilisée
    • Version du pilote installé, date du pilote
    • Usage GPU % et température en temps réel
    • Résolutions supportées

  RAM
    • Capacité totale installée / utilisée / disponible
    • Nombre de barrettes, type (DDR4/DDR5), fréquence (MHz)
    • Configuration (Single/Dual/Quad Channel)
    • Slots utilisés vs disponibles

  Stockage (Disques & Volumes fusionnés)
    • Tous les disques physiques : modèle, type (SSD/HDD/NVMe), capacité
    • Partitions/volumes : lettre, taille, espace libre, système de fichiers
    • Santé SMART (si disponible) : température disque, secteurs réalloués
    • Taux d'occupation en pourcentage et barre visuelle

  Écrans
    • Résolution active et résolutions disponibles
    • Fréquence de rafraîchissement (Hz)
    • HDR activé/désactivé, type de connexion (HDMI, DP, VGA…)
    • Modèle du moniteur (si EDID disponible)

  Batterie (PC portables uniquement)
    • Capacité de design vs capacité actuelle → calcul d'usure (%)
    • Charge actuelle (%) et état (en charge / sur batterie / pleine)
    • Temps restant estimé
    • Cycles de charge (si disponible)

  Périphériques
    • Périphériques USB connectés (nom, fabricant, VID/PID)
    • Imprimantes installées (nom, état, partagée ou non)
    • Périphériques audio (haut-parleurs, micros, état par défaut)

  ── [SYSTÈME] ──────────────────────────────────────────────────────────────

  Windows
    • Version complète : Windows 10/11, édition (Home/Pro/Enterprise)
    • Build number et révision (ex: 26100.8037)
    • Statut d'activation (Activé / Non activé / KMS / Retail)
    • Date d'installation, dernière mise à jour

  BIOS/UEFI
    • Fabricant, version et date du BIOS
    • Mode démarrage : UEFI ou Legacy BIOS
    • Secure Boot activé/désactivé
    • Temps de boot Windows (en secondes)

  Processus
    • Liste complète de tous les processus actifs
    • PID, nom, chemin de l'exécutable
    • CPU % et RAM utilisée par processus
    • Action : terminer un processus (kill) directement depuis la liste

  Services
    • Tous les services Windows (200+)
    • État : En cours / Arrêté / En pause
    • Type de démarrage : Automatique / Manuel / Désactivé
    • Actions : Démarrer / Arrêter / Redémarrer un service

  Démarrage
    • Tous les programmes qui se lancent au démarrage Windows
    • Sources : registre Run, dossier Startup, tâches planifiées
    • Action : Activer / Désactiver chaque entrée

  Tâches planifiées
    • Liste des tâches Windows Task Scheduler
    • État, déclencheur, dernière exécution, prochaine exécution
    • Actions : Activer / Désactiver / Supprimer

  ── [RÉSEAU] ───────────────────────────────────────────────────────────────

  Interfaces réseau
    • Toutes les cartes réseau (filaire + WiFi + virtuel)
    • Adresse IP (IPv4 + IPv6), masque, passerelle par défaut
    • Adresse MAC, vitesse de connexion
    • État : Connecté / Déconnecté / Désactivé

  WiFi
    • Réseau actuellement connecté (SSID, signal, sécurité, bande)
    • Liste des réseaux disponibles à portée (SSID + signal dBm)
    • Profils WiFi sauvegardés sur ce PC

  Connexions TCP/UDP actives
    • Toutes les connexions réseau actives et en écoute
    • Adresse locale (IP:port), adresse distante, état TCP
    • PID du processus propriétaire de chaque connexion

  Bluetooth
    • État de l'adaptateur Bluetooth
    • Périphériques appairés (voir section dédiée)

  DNS
    • Serveurs DNS configurés par interface
    • Test de résolution DNS en direct

  ── [SÉCURITÉ] ─────────────────────────────────────────────────────────────

  Antivirus
    • État de Windows Defender (actif / désactivé / à jour)
    • Date de la dernière mise à jour des définitions
    • Date et résultat du dernier scan

  Pare-feu
    • État du pare-feu Windows par profil (Domaine / Privé / Public)
    • Règles actives (nombre entrant / sortant)

  Comptes utilisateurs
    • Tous les comptes locaux Windows
    • Type (Administrateur / Utilisateur standard), état (Activé/Désactivé)
    • Appartenance aux groupes (Administrateurs, Utilisateurs…)

  Partages réseau
    • Dossiers partagés sur ce PC (nom du partage, chemin, permissions)
    • Partages administratifs (C$, ADMIN$…)

  Certificats
    • Certificats installés dans les stores Windows
    • Émetteur, sujet, date d'expiration, validité

  Persistance (détection menaces)
    • Clés de registre Run / RunOnce listées
    • Détection d'entrées suspectes (chemin inhabituel, fichier absent)

  ── [PILOTES] ──────────────────────────────────────────────────────────────

  Pilotes système
    • Tous les pilotes chargés par Windows
    • Nom, fabricant, version, date, état (en cours / arrêté)
    • Signalement des pilotes non signés (potentiellement dangereux)

  Analyse BSOD (voir section 4.4 pour le détail complet)

  ── [SCAN] ─────────────────────────────────────────────────────────────────

  Scan total
    • Analyse globale du système en une passe
    • Vérifie : Windows Update, intégrité système (SFC/DISM), sécurité,
      espace disque, santé RAM, processus suspects
    • Rapport avec score et recommandations prioritaires

  Scan Antivirus
    • Scan rapide ou complet via Windows Defender
    • Résultats et historique des détections

    05. Désactiver les conseils et astuces Windows (Tips)
    06. Supprimer les applications UWP bloatware (CandyCrush, Netflix préinstallés…)
    07. Désactiver Xbox Game Bar
    08. Désactiver la recherche Bing dans le menu Démarrer
    09. Désactiver les Widgets Windows 11
    10. Désactiver les publicités dans l'interface Windows
    11. Désactiver l'historique des activités
    12. Désactiver l'assistance à distance (Remote Assistance)

  Assistant IA local via Ollama — fonctionne sans connexion internet.

  FONCTIONNALITÉS DE L'AGENT :
    • Diagnostic assisté — l'IA analyse les données de votre système et répond
      à des questions comme "Pourquoi mon PC est-il lent ?"
    • Nettoyage guidé — recommandations personnalisées de nettoyage
    • Analyse sécurité — détection d'anomalies réseau, processus suspects
    • Optimisation — suggestions de réglages adaptées à votre configuration
    • Chat en langage naturel — posez vos questions librement

  PRÉREQUIS POUR UTILISER L'AGENT IA :
    1. Téléchargez et installez Ollama : https://ollama.ai
    2. Ouvrez un terminal et téléchargez un modèle :
         ollama pull llama3:8b          (recommandé, ~5 GB)
         ollama pull mistral:7b          (alternative plus légère)
    3. Démarrez le serveur Ollama : ollama serve
       (ou laissez-le démarrer automatiquement au login Windows)
    4. Dans NiTriTe > Paramètres > IA : vérifiez l'URL (http://localhost:11434)

  CONFIGURATION :
    • URL du serveur : modifiable dans Paramètres > IA
    • Modèle : choisissez parmi les modèles Ollama installés
    • Température : 0.0 (déterministe) à 1.0 (créatif) — 0.7 par défaut

  UTILISATION :
    1. Assurez-vous qu'Ollama tourne (icône dans la barre des tâches)
    2. Ouvrez Agent IA dans NiTriTe
    3. Tapez votre question et appuyez sur Entrée
    4. L'IA répond en tenant compte de l'état réel de votre système

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ "localhost a refusé la connexion" / page blanche avec ERR_CONNECTION    │
  └─────────────────────────────────────────────────────────────────────────┘
  CAUSE    : Vous utilisez un exe de développement (dossier src-tauri\target\)
             sans frontend embarqué, ou le frontend Vue n'a pas été compilé.
  SOLUTION : Utilisez UNIQUEMENT les exes dans le dossier release\ :
               release\Nitrite_v26.32.0_portable.exe
               release\Nitrite_v26.32.0\ (Version 2)
               release\Nitrite_v26.32.0_full.exe (Version 3 — extrait d'abord)
             Les fichiers dans src-tauri\target\release\ sont réservés au
             développement et ne fonctionnent pas comme portables.

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ L'application ne démarre pas (rien ne se passe au double-clic)          │
  └─────────────────────────────────────────────────────────────────────────┘
  CAUSE    : UAC refusé, ou WebView2 Runtime absent (rare sur Windows 11).
  SOLUTION : • Clic droit sur Nitrite.exe → "Exécuter en tant qu'administrateur"
             • Si le problème persiste : téléchargez WebView2 Runtime sur
               microsoft.com (recherchez "WebView2 Runtime download")

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ Les Apps Portables / Drivers / Scripts ne s'ouvrent pas ou sont absents │
  └─────────────────────────────────────────────────────────────────────────┘
  CAUSE    : Vous utilisez la Version 1 (exe seul) sans les dossiers
             logiciel\, Drivers\ et Script Windows\ à côté.
  SOLUTION : Utilisez la Version 2 ou 3 qui inclut tous ces dossiers.
             Ou placez manuellement les dossiers logiciel\, Drivers\ et
             "Script Windows\" dans le même répertoire que Nitrite.exe.

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ L'Agent IA ne répond pas / "Impossible de contacter Ollama"             │
  └─────────────────────────────────────────────────────────────────────────┘
  CAUSE    : Ollama n'est pas installé, pas démarré, ou le modèle est absent.
  SOLUTION :
    1. Installez Ollama depuis https://ollama.ai
    2. Ouvrez un terminal et téléchargez un modèle :
         ollama pull llama3:8b
    3. Démarrez Ollama : ollama serve
       (vérifiez qu'il tourne sur http://localhost:11434)
    4. Dans NiTriTe → Paramètres → IA :
       vérifiez l'URL (http://localhost:11434) et le nom du modèle

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ "Accès refusé" sur certaines fonctions                                  │
  └─────────────────────────────────────────────────────────────────────────┘
  CAUSE    : NiTriTe n'a pas été lancé avec les droits Administrateur.
  SOLUTION : Clic droit sur Nitrite.exe → "Exécuter en tant qu'administrateur"
             Normalement, le bouclier UAC s'affiche au premier lancement
             et NiTriTe s'exécute déjà en admin. Si ce n'est pas le cas,
             vérifiez que votre compte Windows est bien Administrateur.

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ L'interface est blanche ou ne charge pas après le démarrage             │
  └─────────────────────────────────────────────────────────────────────────┘
  CAUSE    : Conflit avec un proxy, VPN ou antivirus bloquant WebView2.
  SOLUTION : • Ajoutez Nitrite.exe en exception dans votre antivirus
             • Désactivez temporairement le VPN/proxy
             • Réessayez en mode administrateur

  ┌─────────────────────────────────────────────────────────────────────────┐
  │ La Version 3 (SFX) prend très longtemps à extraire                      │
  └─────────────────────────────────────────────────────────────────────────┘
  CAUSE    : Normal — 2.4 GB de données sont extraits et décompressés.
  SOLUTION : Soyez patient. La durée varie selon la vitesse de votre support :
               SSD NVMe    : 3-5 minutes
               SSD SATA    : 5-8 minutes
               HDD          : 10-20 minutes
               Clé USB 3.0 : 10-15 minutes
               Clé USB 2.0 : 30-60 minutes (déconseillé pour l'extraction)
             L'extraction n'est faite QU'UNE SEULE FOIS.
             Les lancements suivants depuis le dossier extrait sont immédiats.
